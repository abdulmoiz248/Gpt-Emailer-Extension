How to Use

1. Set up your email accounts above.
2. When chatting with ChatGPT, use the keyword sendEmail followed by a JSON code block: 

sendEmail
```json
{
  "to": "recipient@example.com",
  "subject": "Your Subject",
  "body": "Your email content here"
}
```

3. The extension will detect this format and prompt you to choose which account to send from.

Note: For Gmail accounts, you'll need to use an App Password. Learn how to create one.
